# rs_rotorquant_codec

Rust crate in the gfxATOM submodule workspace. RotorQuant KV cache compression codec implementing PlanarQuant (2D Givens rotations, 3-bit or 4-bit) and IsoQuant (4D quaternion rotations, 3-bit or 4-bit) with Lloyd-Max codebook training.
